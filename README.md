# 超市管理系统 java76

## 项目概述

本项目是基于SpringBoot和Vue的前后端分离的超市管理系统。该系统集成了商品管理、仓库管理、销售管理、人事管理、会员管理等功能模块，适用于线上物品平台和超市信息管理。

## 技术栈

- **开发语言**：Java
- **后端框架**：SpringBoot
- **前端框架**：Vue
- **数据库**：Mysql
- **系统环境**：Windows，Mac
- **开发环境**：IDEA、JDK1.8

- **个人中心**

  - 修改密码
